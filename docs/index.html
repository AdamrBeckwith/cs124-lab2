Creating New checklist Items

This is the initial base of the image. You can hit the plus button (or anywhere on "new item") to add an item when the sheet is blank

Once you hit the button, the element is added. It is added in the location where the add item was and is unchecked (not completed ) by default. As a result the add item butto is moved downward. The add item button is always on the bottom the list. New items always take its place

If you want to add another item then tap the add button again .This will create a new item directly below the lowest item on the checklist.

Marking Items as complete.

Suppose you have a task that you have completed (like calling mom). In order to mark it as complete, simply tap the checkbox and the box will change into a check.

Here we have complete the mom call. The check is now signature blue and we know it is complete :D

Renaming elements

Suppose you misenter a list item and want to change it. Double tap to cause the caret to appear on the text. Now you can edit

See the cursor, Now use your phones keyboard to update your checklist item like show below

Show/Hide Completed items

Suppose you had a complex of things to do, You dont want to get rid of it permanently but wanted to focus on the incomplete ones. tap the hide button to hide all selected items

Now only uncompleted items show. If you mark a item as completed like this it will disapear. Now tap the show button to bring the completed items back, they stay completed.

You can press the show/ hide button as much as you want!

Delete Completed Items

Lets say you want to delete some items permantly. Tap the Trash button. Note this will delete all checked items :D

No mistakes here! The app will check that "You are sure?" before you complete the delete. tap again to delete everything. This stays red until you tap somewhere else

No regrets! The items are deleted forever and cannot be recovered. You can now start checking new completed items.
